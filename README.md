# Hey, I'm Guillermo 👋

I'm a developer from Spain focused on AI, data engineering, and building things that actually work in production.

My background is a mix of backend development (worked at Endesa automating data workflows), mathematics, art, and a specialization in AI & Big Data where I got to fine-tune LLMs and build distributed systems on real Hadoop clusters. I enjoy optimizing systems and solving challenges.

Lately I've been deep into LLM pipelines. I co-built a medical chatbot with a real hospital using RAG, TEI and TGI, and I'm applying what I learned in new projects. I'm also AWS certified and spending time with LangGraph and n8n for agentic workflows.

- 🔭 Currently building a RAG chatbot coach with AI generated voice, inspired by Earl Nightingale
- 🌱 Working through AWS Cloud Architecting and exploring LangGraph for multi-agent systems
- 💬 Happy to talk about Hadoop clusters, YOLO models, fine-tuning Qwen, or data pipelines
- 🤝 Open to collaborate on projects where data actually matters: health, energy, automation...
- ⚡ I moved to Stuttgart knowing zero German and hit B2 in under a year. Same approach to everything I learn
